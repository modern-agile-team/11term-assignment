구동영상

## 프로젝트 설명
Express와 MySQL을 이용한 Todo List CRUD 웹 애플리케이션

## 기술 스택
- Front-End: HTML, CSS, JavaScript (fetch)
- Back-End: Node.js, Express
- Database: MySQL

## 주요 기능
- Todo 목록 조회
- Todo 추가
- Todo 수정 및 완료 처리
- Todo 삭제

## 실행 방법
1. MySQL 실행 및 DB 생성
2. .env 파일 설정
3. 서버 실행
```bash
npm install
npm run dev
